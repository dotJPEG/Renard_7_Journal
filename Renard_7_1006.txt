What are methods?
Methods are like mini programs within a program that exist simply to make running the same command quick and easy.

What are classes?
Classes are a section in code that acts as one program, everything within the class effects only the other things in the class.

What is an instance?
An instance is a class that you run digitally that isnt in the code, when you run an instance you allocate memory for a new object and returning a reference to that memory

What are class methods?
class methods are methods that are called on a class.
